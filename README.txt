This plugin adds a search item in the nav menu of your choice

== Description ==

Using TB Search in Menu you can easily add a search item in your menu that will show a modern search form when clicked. All you need to do is activate the plugin and choose the menu location where you want your search item to be shown.
Brought to you by [theme.blue](http://theme.blue)

== Installation ==

1. Go to Plugins > Add New and search for TB Search in Menu. Install and activate it.
2. Go to Appearance > Customize > TB Search in Menu and select the menu location you want. That's it.
3. Optional: Play with the settings located under the same tab in the Customizer.
